# FoodLens
Image Based Food Recognition &amp; Recipe Info

It is a project focused on classifying 80 different food items from images using deep learning techniques.

## Overview

This repository contains the code and resources for a deep learning project focused on food classification. The project aims to accurately classify various types of food items from images using deep convolutional neural networks.

## Getting Started

1. Clone the repository: `git clone https://github.com/ABHISHEKSUBHASHSWAMI/FoodLens.git`
2. Set up a Python environment (preferably a virtual environment) and install the required packages.

## Dataset
The dataset used for this project contains 4000+ images of 80 different food items, resized and preprocessed to a uniform size of 256 x 256 pixels.
[Download](https://www.kaggle.com/datasets/abhisheksubhashswami/indian-food)
